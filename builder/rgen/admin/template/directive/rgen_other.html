Animation settings

Animation working based on animate.css (https://daneden.github.io/animate.css/)

Below is example to apply animation manually from source view option.

data-animate-in="fadeIn|1" or

data-animate-in="fadeIn"

Attribute first part is animation class name "fadeIn" and second part after "|" is delay value. You can also apply only CSS animation class if you do not wish to set delay.
